# artcode Bass

## 概要

**artcode Bass** は、太く力強いベースサウンドに特化したソフトウェアシンセサイザーです。VST3/CLAP形式で、あらゆるDAW（Cubase、Ableton Live、FL Studio、Logic Pro、Reaper など）で使用できます。

EDM、ダブステップ、ヒップホップ、テクノなど幅広いジャンルで活躍するサブベース、ウォブルベース、アシッドベースなどのサウンドを簡単に作成できます。

---

## 特徴

### 🎛️ ベース特化設計
- **サブオシレーター**: メインオシレーターの1オクターブ下でサイン波を生成し、低域を強化
- **モノフォニック**: ベースに最適化された単音モード（常時有効）
- **-12半音トランスポーズ**: デフォルトで1オクターブ下の音域

### 🔊 パワフルなサウンドエンジン
- **デュアルオシレーター**: 4種類の波形（Sine / Saw / Square / Triangle）
- **8ボイスユニゾン**: スプレッド調整で太いサウンドを実現
- **ステートバリアブルフィルター**: LP/HP/BP、12dB/24dBスロープ切り替え
- **4種類のドライブ**: Soft / Hard / Tube / Fuzz
- **ローシェルフEQ**: 100Hz以下を最大+12dBブースト

### 🎹 演奏機能
- **ポルタメント**: 滑らかなピッチ変化
- **アルペジエーター**: DAWテンポ同期、4つのモード、最大4オクターブ
- **LFO**: フィルター、ピッチ、アンプをモジュレーション

### 🎨 直感的なインターフェース
- **オレンジ色のアクセントカラー**: 視認性の高いデザイン
- **30種類のファクトリープリセット**: すぐに使えるサウンド
- **カテゴリー分類**: Sub / Fat / Acid / Wobble / Growl / Clean

- **DAW**: VST3またはCLAP対応

ダウンロードしたプラグインファイルを以下の場所にコピーしてください：

## パラメータガイド

### OSC（オシレーター）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **OSC1 Wave** | オシレーター1の波形 | Sin/Saw/Sqr/Tri | Saw |
| **OSC1 Det** | オシレーター1のデチューン | ±100 cents | 0 |
| **OSC2 Wave** | オシレーター2の波形 | Sin/Saw/Sqr/Tri | Saw |
| **OSC2 Det** | オシレーター2のデチューン | ±100 cents | +7 |
| **OSC2 Mix** | OSC2のミックス量 | 0〜100% | 50% |
| **Sub Vol** | サブオシレーター音量 | 0〜100% | 50% |

**💡 ヒント:**
- サブベースには **Sub Vol を高め**に設定
- OSC2のデチューンを少しずらすと太いサウンドに
- Sin波形はクリーンなサブベース、Saw波形はアグレッシブなサウンド

---

### UNISON（ユニゾン）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **Unison** | ユニゾンボイス数 | 1〜8 | 4 |
| **Spread** | デチューンスプレッド | 0〜100% | 25% |

**💡 ヒント:**
- ボイス数を増やすと厚みが増すがCPU負荷も上がる
- Spread を上げるとコーラスのような広がりに
- クリーンなサブベースには Unison: 1 がおすすめ

---

### FILTER（フィルター）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **Cutoff** | カットオフ周波数 | 20Hz〜20kHz | 600 Hz |
| **Reso** | レゾナンス | 0〜99% | 40% |
| **Flt Env** | フィルターエンベロープ量 | -100〜+100% | 7% |
| **Type** | フィルタータイプ | LP/HP/BP | LP |
| **Slope** | フィルタースロープ | 12dB/24dB | 24dB |

**💡 ヒント:**
- アシッドベースには **高いReso** と **高いFlt Env** を設定
- 24dBスロープはより急峻なカットオフでパンチのある音に
- BP（バンドパス）は中域を強調したサウンドに

---

### DRIVE（ドライブ）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **Drive** | ドライブ量 | 0〜100% | 10% |
| **Type** | ドライブタイプ | Soft/Hard/Tube/Fuzz | Tube |
| **Low Boost** | 低域ブースト（100Hz） | 0〜100% (+12dB) | 50% |

**ドライブタイプの特徴:**
| タイプ | 特徴 |
|--------|------|
| **Soft** | 穏やかなサチュレーション、クリーンに近い |
| **Hard** | アグレッシブなクリッピング |
| **Tube** | 真空管アンプ風の温かみ |
| **Fuzz** | 激しい歪み、ダブステップ向き |

**💡 ヒント:**
- Low Boost は808風サブベースに効果的
- 音割れする場合は Drive を下げてください
- Tube はほとんどのベースサウンドに合う

---

### ADSR（エンベロープ）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **Attack** | アタックタイム | 1ms〜5s | 5ms |
| **Decay** | ディケイタイム | 1ms〜5s | 150ms |
| **Sustain** | サステインレベル | 0〜100% | 70% |
| **Release** | リリースタイム | 1ms〜10s | 150ms |

**💡 ヒント:**
- 808ベースは **Sustain: 0%** で減衰する音に
- パッド的なベースは長い Attack と Release を
- スタッカートには短い Release を設定

---

### LFO セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **Rate** | LFO速度 | 0.01〜50 Hz | 2 Hz |
| **Depth** | LFO深さ | 0〜100% | 0% |
| **Wave** | LFO波形 | Sin/Saw/Sqr/Tri | Sin |
| **Target** | モジュレーション先 | Pitch/Filter/Amp | Filter |
| **Porta** | ポルタメントタイム | 0〜1s | 7ms |

**💡 ヒント:**
- ウォブルベースには **Target: Filter**、**Rate: 2〜8Hz**
- ビブラートには **Target: Pitch**、**Depth: 小さめ**
- トレモロには **Target: Amp**

---

### ARP（アルペジエーター）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **ON/OFF** | アルペジエーター有効化 | ON/OFF | OFF |
| **Mode** | 演奏パターン | Up/Down/UpDn/Rnd | Up |
| **Speed** | 音符の長さ | 1/4, 1/8, 1/16, 1/32 | 1/8 |
| **Oct** | オクターブ範囲 | 1〜4 | 1 |

**モードの説明:**
| モード | 動作 |
|--------|------|
| **Up** | 低い音から高い音へ |
| **Down** | 高い音から低い音へ |
| **UpDn** | 上昇→下降を繰り返し |
| **Rnd** | ランダム |

**💡 ヒント:**
- アルペジエーターはDAWのテンポに自動同期
- ベースラインの作成に活用できる
- Oct を増やすとより広い音域をカバー

---

### FX（エフェクト）セクション

| パラメータ | 説明 | 範囲 | デフォルト |
|-----------|------|------|----------|
| **Delay** | ディレイミックス | 0〜100% | 0% |
| **Time** | ディレイタイム | 50ms〜1s | 300ms |
| **FB** | ディレイフィードバック | 0〜95% | 40% |
| **Reverb** | リバーブミックス | 0〜100% | 0% |
| **Size** | リバーブサイズ | 10〜99% | 50% |

**💡 ヒント:**
- ベースはドライ（エフェクトなし）が基本
- ダブ系にはディレイを少しだけ
- リバーブをかけすぎると低域が濁る

## トラブルシューティング

### 音が割れる
- **Drive** を下げてください（10〜20%程度が目安）
- **Vol**（マスターボリューム）を下げてください

### 音が出ない
- DAWのトラックがミュートになっていないか確認
- MIDIキーボード/ノートが正しく入力されているか確認

### アルペジエーターの速度がおかしい
- DAWのテンポ設定を確認してください
- アルペジエーターはDAWテンポに同期します

### CPU負荷が高い
- **Unison** のボイス数を減らす
- **Reverb** のサイズを小さくする
